# 📚 스마트 교실 비품 대여 관리 시스템 (Smart Rental System)

QR 코드와 실시간 데이터베이스를 활용하여 학교 교실 내 비품 대여를 자동화하고, 게임 요소를 도입하여 학생들의 올바른 대여 습관을 장려하는 종합 관리 솔루션입니다.

---

## ✨ 핵심 기능

### 👨‍🏫 교사 (관리자) 기능
- **비품 통합 관리**: 사진을 포함한 비품 등록, 수정, 삭제 및 관리.
- **실시간 대시보드**: 학급 전체 대여 현황 및 미반납 내역 실시간 모니터링.
- **가입 승인 시스템**: 승인된 학생만 서비스를 이용할 수 있도록 보안 강화.
- **상태 신고 확인**: 학생들이 접수한 비품 파손/부족 신고 실시간 수신 및 조치.
- **데이터 분석 및 내보내기**:
  - **대여 통계**: 인기 비품 순위 시각화 차트 제공.
  - **엑셀 내보내기**: 모든 대여 이력을 CSV 파일로 추출하여 행정 처리 지원.
- **학생 포인트 관리**: 학생별 상벌점 부여 및 수동 포인트 조정.

### 🎒 학생 기능
- **간편 대여/반납**: 내장 카메라로 QR 코드를 스캔하여 1초 만에 대여 및 반납 처리.
- **스마트 알림 배너**: 반납 기한 임박(내일) 또는 연체 시 상단에 경고 배너 자동 표시.
- **자동 대기열(Waitlist)**: 품절된 비품에 대기 신청을 하면, 반납 시 즉시 알림 제공.
- **상태 신고 시스템**: 대여 중인 물품의 문제점(파손, 소모품 부족 등)을 교사에게 즉시 보고.
- **개인 대여 관리**: 현재 대여 중인 물품 리스트 및 개인 포인트 히스토리 조회.

---

## 🏆 게이미피케이션 (Gamification)

학생들의 동기 부여를 위해 다음과 같은 보상 체계가 구축되어 있습니다.

### 1. 등급 시스템
포인트에 따라 5단계 등급과 아이콘이 부여되며, 등급별로 대여 한도가 차등 적용됩니다.
- **Gold (🥇)**: 300점 이상 / 최대 10개 대여
- **Silver (🥈)**: 100점 이상 / 최대 5개 대여
- **Bronze (🥉)**: 기본 등급 / 최대 3개 대여
- **Warning (⚠️)**: 벌점 누적 시 / 최대 1개 대여
- **Banned (🚫)**: 서비스 이용 제한

### 2. 업적 뱃지
특정 조건을 만족하면 프로필에 특별한 뱃지가 표시됩니다.
- 🦅 **얼리버드**: 조기 반납 보너스 3회 이상 획득
- ✨ **깔끔쟁이**: 대여 5회 이상 + 연체 기록 없음
- 📚 **독서왕**: 도서 카테고리 3회 이상 대여

## 🎨 디자인 및 환경
- **프리미엄 UI/UX**: 애니메이션 스플래시 화면 및 Glassmorphism 스타일 적용.
- **다크/라이트 모드**: 사용자의 환경에 맞는 테마 토글 기능 (설정 저장).
- **반응형 웹**: 스마트폰 카메라 스캔에 최적화된 모바일 퍼스트 디자인.

---

## 🛠 기술 스택
- **언어**: HTML5, Vanilla CSS, JavaScript (ES6+)
- **백엔드**: Firebase Firestore (Real-time DB / Auth)
- **도구**: Html5Qrcode (QR Scanning), QRServer API (QR Generation)

---

## 🚀 설치 및 시작
1. 본 저장소를 `Git Clone` 합니다.
2. `Firebase Console`에서 프로젝트를 생성하고, `app.js` 상단의 `firebaseConfig`를 본인의 설정값으로 교체합니다.
3. `index.html`을 실행합니다.
4. (최초 실행 시) 교사 계정으로 가입하면 기본 비품 데이터가 자동으로 생성됩니다.
