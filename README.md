# 📚 교실 비품 대여 관리 시스템 (Smart Rental System)

QR코드를 활용하여 학교 교실의 비품을 효율적으로 관리하고 대여할 수 있는 웹 애플리케이션입니다.

## ✨ 주요 기능

### 👨‍🏫 교사 (관리자) 모드
- **비품 관리**: 비품 등록, 수정, 삭제 및 QR코드 생성/인쇄
- **실시간 대여 현황**: 현재 어떤 학생이 무엇을 빌려갔는지 대시보드에서 한눈에 확인
- **대여 통계**: 가장 많이 대여되는 비품을 막대 그래프로 확인
- **상벌점 시스템**: 학생별 점수 관리 및 등급(Gold, Silver, Bronze 등) 부여

### 🎒 학생 모드
- **QR 스캔 대여/반납**: 카메라로 비품 QR을 스캔하여 간편하게 대여 및 반납
- **내 대여 목록**: 내가 빌린 물품과 반납 기한 확인
- **포인트 시스템**: 기한 내 반납 시 포인트 획득 및 등급 상승

### 🎨 디자인 및 UX
- **스플래시 화면**: 앱 시작 시 세련된 로고 연출
- **반응형 디자인**: 모바일과 PC 어디서든 최적화된 화면 구성
- **다크 모드 테마**: 눈이 편안한 프리미엄 다크 UI 적용

## 🛠 기술 스택
- **Frontend**: HTML5, CSS3, JavaScript (Vanilla JS)
- **Backend**: Firebase Firestore (Real-time Database)
- **Library**: Html5Qrcode (QR Scanning)
